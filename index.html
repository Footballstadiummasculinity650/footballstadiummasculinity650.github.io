🛣️ MRoadGen - Build complex road networks on terrain

Visit repository to download MRoadGen

📋 Overview

MRoadGen provides a set of geometry nodes for Blender users. It automates the process of drawing roads over uneven land. You save time by skipping manual modeling. The tool handles high-detail terrain meshes. You keep full control over the road path and shape.

⚙️ System Requirements

Ensure your computer meets these standards before you start:

  • Blender Version: Blender 4.0 or newer.
  • Operating System: Windows 10 or 11.
  • Memory: 16GB of RAM is recommended for large scenes.
  • Graphics Card: A dedicated GPU with at least 4GB of video memory.

📥 Installation Guide

Follow these steps to add the tool to your Blender workflow:

  1. Visit the official download page to obtain the files.
  2. Download the ZIP file to a folder on your computer.
  3. Extract the contents of the ZIP file.
  4. Open Blender.
  5. Go to File, select Append, and navigate to the extracted file.
  6. Select the Geometry Node group and move it into your scene.

🛠️ How to Use

Once you load the node group, you attach it to your terrain object. Use the modifier panel to adjust settings. You define the path by placing points on the surface. The tool automatically projects the road onto the terrain. Adjust the width, curb height, and material settings using the sliders provided in the node interface.

❓ Frequently Asked Questions

Can I use this for animation? Yes. Because it uses geometry nodes, the road updates in real-time as you move your control points.

Does it support custom textures? Yes. The system includes material slots for the road and shoulder edges.

Is it heavy on my computer? The system is designed for high-poly meshes but you should keep your base terrain topology reasonable for the best performance.
